# Timeline Content

This file contains all your career milestones. Edit the frontmatter above to add, remove, or modify entries.

## How to Edit

1. Each milestone is defined in the `milestones` array in the frontmatter (between the `---` markers)
2. To add a new milestone, copy an existing one and modify the fields
3. To remove a milestone, delete the entire entry
4. The timeline will automatically sort by `sortDate` in reverse chronological order

## Field Guide

- **id**: Unique identifier (can be anything, like "job-google-2024")
- **date**: Display date shown on timeline (e.g., "January 2024", "Summer 2023")
- **sortDate**: ISO date for sorting (YYYY-MM-DD format)
- **title**: Milestone title/headline
- **description**: Rich HTML content (use paragraph tags, links, etc.)
- **tags**: Optional array of tags/categories
- **media**: Optional array of images, videos, or embeds

### Media Types

**Image:**
```yaml
- type: "image"
  src: "/images/photo.jpg"
  alt: "Description"
  caption: "Optional caption"
```

**Local Video:**
```yaml
- type: "video"
  src: "/videos/demo.mp4"
  caption: "Optional caption"
```

**YouTube:**
```yaml
- type: "embed"
  platform: "youtube"
  url: "https://www.youtube.com/watch?v=VIDEO_ID"
```

**X (Twitter):**
```yaml
- type: "embed"
  platform: "x"
  url: "https://x.com/username/status/TWEET_ID"
```

**Instagram:**
```yaml
- type: "embed"
  platform: "instagram"
  url: "https://www.instagram.com/p/POST_ID/"
```
